# 📓 Learning Journal

A coding-focused learning journal built to track progress, organize concepts, and reflect on the learning process through structured entries.

## 🚀 Overview
The Learning Journal allows users to create, manage, and search entries related to their coding journey. It focuses on simplicity, usability, and clear organization, making it easy to document growth over time.

## ✨ Features
- Add and save journal entries with title, date, and content  
- Categorize entries by coding topic  
- Search entries by title, content, or topic  
- Automatically sort entries by most recent date  
- Delete entries  
- Persistent data storage using localStorage  
- Responsive design for mobile and desktop  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  
- LocalStorage
